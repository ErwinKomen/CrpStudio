About Corpus Studio Web

Text collections on the internet are no novelty, and there are lots of programs facilitating access to them. The number of texts that come with syntactic annotation is slowly increasing, and working with these texts has been a challenge. Corpus Studio Web provides comprehensive search capabilities to these texts (since the search language works on the xml texts directly). The basic component to these searches is the 'Corpus Research Project' (a CRP). Such a CRP holds the essential parts of a corpus research task, such as:

  • the queries that need to be used
  • the order (and hierarchy) of the queries
  • the texts on which the queries are made
  • meta information (author, date, version, description)

Where Corpus Studio facilitated the creation and running of CRPs in a Windows environment, working on whatever texts the user might have available, the web-based version of Corpus Studio is platform independent and keeps the corpora in one central place.

The Corpus Studio web version you are now using provides access to the corpora available on the Corpora page.

There are two options to use the Corpus Studio web version for other corpora:

  • Request permission to add the corpus to this Corpus Studio web version (see contact details)
  • Ask the local technical department to install the Corpus Studio web version that is available on GitHub (see details)
